# Font Collection

A curated collection of programming and design fonts.

## Fonts Included

- **Fira Code v6.2** - Monospaced font with programming ligatures
- **JetBrains Mono 2.304** - Modern monospaced font designed for developers
- **IBM Plex Mono** - IBM's corporate typeface, monospaced variant
- **Inter** - Versatile sans-serif font optimized for user interfaces
- **Cabin, Cinzel Decorative, Comfortaa, Cormorant Garamond, Crimson Text** - Collection of display and text fonts
- **Lora, Nunito, Playfair Display, Poppins, Quicksand** - Modern web fonts for various design needs

## Usage

Extract the desired font archive and install the font files (.ttf, .otf) by double-clicking or using your system's font manager.
